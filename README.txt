# Treatment Dashboard – Power BI Project

## Project Overview
This Power BI dashboard provides a comprehensive analysis of hospital treatment performance, patient outcomes, admissions, and billing information.  
The dashboard is designed to help healthcare management teams monitor treatment trends, evaluate outcomes, and make data-driven decisions.

---

## Dashboard Features

### KPIs
- Total Chemotherapy Bills
- Total Patients by Treatment Type

### Filters / Slicers
The dashboard includes interactive filters for:
- Outcome
- Quarter
- Year
- Month
- Treatment Type

### Visualizations
#### 1. Number of Patients Per Treatment Type
Displays the number of patients across:
- Chemotherapy
- Medication
- Immunotherapy
- Physical Therapy
- Radiation
- Surgery

#### 2. Outcomes Per Treatment
Compares treatment outcomes including:
- Recovered
- Death
- Complication

#### 3. Total Bills Per Treatment
Pie chart showing treatment billing distribution.

#### 4. Admissions Per Month
Monthly patient admission trend analysis.
